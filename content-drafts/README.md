# Content drafts

Drafts in progress. Vini reviews here before pieces are moved to `/web/src/content/articles/` (or `articles-ar/`) for publication.

## Folders

- `inbox/` — fresh drafts from the Editor, awaiting QA
- `approved/` — Vini-approved, awaiting Web Developer to move them into the site
- `archive/` — drafts Vini rejected, kept for reference

## Files

- `calendar.md` — the rolling editorial calendar (Editor maintains, Manager confirms weekly)
- `curated-queue.md` — the queue of primary-source links the Editor has flagged for upcoming curated short pieces

## Draft naming convention

`YYYY-MM-DD-<country-slug>-<short-slug>.md`

Example: `2026-05-26-sierra-leone-mende-kindergartens.md`
