# Social drafts

Maintained by the Growth agent. **Vini sends — Growth never posts.** Every external message is reviewed before it leaves.

## Folders

- `instagram/` — caption drafts (filename: `YYYY-MM-DD-<slug>.md`)
- `newsletter/` — newsletter edition drafts (filename: `YYYY-MM-DD-issue-NN.md`)
- `community-notes.md` — log of community members worth thoughtful engagement
- `reports/` — monthly metric reports (filename: `YYYY-MM.md`)
