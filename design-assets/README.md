# Design assets

Maintained by the Designer.

## Folders

- `mood-boards/` — initial visual direction options (each board as a standalone HTML preview)
- `system/` — the locked design system once chosen (typography, color, components)
- `hero/` — per-piece hero visuals (`YYYY-MM-DD-<slug>/`)
- `specs/` — specs handed to the Web Developer
- `prompts.md` — log of every AI prompt used to generate imagery, with output reference
